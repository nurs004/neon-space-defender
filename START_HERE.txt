ШАГ 1: ПОДГОТОВКА (5 минут)
═══════════════════════════════════════════════════════════════════════════

1.1. Установите необходимое:

```bash
pip install kivy pillow buildozer cython
```

1.2. Подготовьте проект:

```bash
python prepare_android.py
```

✓ Созданы: data/icon.png, data/presplash.png, PRIVACY_POLICY.txt

═══════════════════════════════════════════════════════════════════════════

ШАГ 2: ЛОКАЛЬНОЕ ТЕСТИРОВАНИЕ (на компьютере)
═══════════════════════════════════════════════════════════════════════════

Тестируйте мобильную версию локально:

```bash
python main.py
```

Играйте и проверяйте:
✓ Все кнопки работают
✓ Нет краш
✓ Графика красивая
✓ Сохраняется рекорд (highscore.json)

═══════════════════════════════════════════════════════════════════════════

ШАГ 3: СБОРКА APK (выберите ОДИН вариант)
═══════════════════════════════════════════════════════════════════════════

### ВАРИАНТ A: GitHub Actions (РЕКОМЕНДУЕТСЯ - БЕСПЛАТНО & ПРОСТО)

1. Создайте новый репозиторий на https://github.com/new
   Название: neon-space-defender

2. Загрузьте код:
```bash
cd my_mini_game
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/neon-space-defender.git
git push -u origin main
```

3. Дождитесь завершения Actions (2-3 минуты)

4. Скачайте APK:
   GitHub → Actions → Build APK → Artifacts → neonspacedefender.apk

✓ Готовый APK получен БЕСПЛАТНО!

---

### ВАРИАНТ B: Локальная сборка на Linux/Mac

```bash
buildozer android debug
# Ждите 15-30 минут...
# APK: bin/neonspacedefender-1.0.0-debug.apk
```

---

### ВАРИАНТ C: Google Colab (облачно, если нет Linux)

1. Откройте https://colab.research.google.com/
2. Создайте новый ноутбук
3. Вставьте этот код:

```python
!apt-get update
!apt-get install -y openjdk-11-jdk build-essential
!pip install buildozer cython

# Загрузите файлы (нажмите upload кнопку)
from google.colab import files
uploaded = files.upload()

# Распакуйте
!unzip my_mini_game.zip
%cd my_mini_game

# Соберите
!buildozer android debug

# Скачайте результат
files.download('bin/neonspacedefender-1.0.0-debug.apk')
```

✓ APK скачана в папку Downloads

═══════════════════════════════════════════════════════════════════════════

ШАГ 4: ТЕСТИРОВАНИЕ НА ТЕЛЕФОНЕ
═══════════════════════════════════════════════════════════════════════════

4.1. Подключите Android телефон через USB:

```bash
# Проверьте подключение
adb devices

# Установите APK
adb install bin/neonspacedefender-1.0.0-debug.apk

# При ошибке - удалите старую версию:
adb uninstall com.spacegames.neonspacedefender
adb install bin/neonspacedefender-1.0.0-debug.apk
```

4.2. Играйте и тестируйте:

✓ Все работает красивого
✓ Нет крашей
✓ На разных телефонах тестируйте
✓ Проверьте батарею (не должно быть горячо)

═══════════════════════════════════════════════════════════════════════════

ШАГ 5: ВЫКЛАДЫВАНИЕ НА GOOGLE PLAY
═══════════════════════════════════════════════════════════════════════════

5.1. Создайте Google Play Developer аккаунт:

- Откройте: https://play.google.com/console
- Нажмите "Create Account"
- Оплатите $25 USD
- Заполните информацию разработчика
  (Имя, почта, адрес - может быть домашний)

⏱️ Займет 30 минут

5.2. Создайте приложение в консоли:

- Google Play Console → "Все приложения" → "Создать приложение"
- Название: "Neon Space Defender"
- Категория: Games → Action
- Выберите язык: Русский

5.3. Загрузите первый выпуск:

В левом меню → "Версии выпуска" → "Создать выпуск"
- Загрузите APK файл
- Версия кода: 1
- Версия выпуска: 1.0.0

5.4. Заполните "Информацию об магазине":

◆ НАЗВАНИЕ (макс 50 символов):
  🌟 NEON SPACE DEFENDER 🌟

◆ КРАТКОЕ ОПИСАНИЕ (макс 80 символов):
  Классический space shooter с неоновым стилем!

◆ ПОЛНОЕ ОПИСАНИЕ (макс 4000 символов):
  Защищайте вашу станцию от врагов в космосе!
  
  ✨ 4 режима стрельбы
  ✨ 5 типов врагов
  ✨ Система волн с боссами
  ✨ Неоновая стилизация 80-х
  
  🎮 Управление: касание для движения
  Постоянная стрельба автоматически
  
  ⭐ Подойдет для любого возраста!

◆ ИКОНКА ПРИЛОЖЕНИЯ:
  Загрузите: data/icon.png (512x512)

◆ СКРИНШОТЫ:
  Минимум 2 скриншота (1080x1920 или 1440x2560)
  Помещайте на главное меню и во время игры

5.5. Рейтинг контента:

В меню "Политика приватности и рейтинги" → "Рейтинг контента"

Заполните анкету:
- Насилие? НЕТ
- Секс? НЕТ
- Другие спорные вещества? НЕТ

Рейтинг: PEGI 3 (для всех возрастов)

5.6. Политика приватности:

В меню → добавьте текст из PRIVACY_POLICY.txt
Или загрузите как ссылку на облако

5.7. ФИНАЛЬНАЯ ПРОВЕРКА:

Обзор в консоли → убедитесь что все заполнено:
✓ Иконка загружена
✓ Скриншоты 2+
✓ Описание полное
✓ Рейтинг установлен
✓ Политика приватности есть
✓ Email контакта заполнен

5.8. ЗАПУСК!

Нажмите "Запустить выпуск"

Google будет проверять 2-4 часа.

После проверки - ГОТОВО! ВЫ В GOOGLE PLAY! 🎉
